Purpose

The X-CUBE-CELLULAR Projects (part of the STM32 Cellular software expansion for STM32Cube)

Provides examples for multiple STM32 boards

  • Discovery L496
  • Discovery L475
  • STWIN (Unofficial)
  • Discovery T1SE (STM32 L462 + ST33 + T1SC (ALT 1250 modem)) (Unofficial)

On Discovery L496 the following modems are supported UG96, BG96, GM01Q and T1SC (Unofficial)

On Discovery L475, only BG96 and GM01Q are supported

On STWIN, BG96 and GM01Q are supported

Available IDEs are IAR, Keil and STM32CubeIDE

Update History

Main Changes

This V5.2.0 adds the unofficial support of Murata T1SE module (Altair ALT1250 modem) on Discovery board (B-L462E-CELL1). it also adds STWIN (STEVAL-STWINKT1) board unofficial support

Previous versions supported the following modems:

  • Quectel module UG96 with Intel modem (3G, won’t be supported anymore in next delivery)
  • Quectel module BG96 with Qualcomm modem (LTE CatM, NBIoT and 2G Fallback)
  • Sequans module GM01Q with Sequans modem (LTE CatM)
  • Murata module T1SC (LTE CatM, NBIoT) (Unofficial)

Main Changes

This V5.1.0 adds the support of Murata T1SC module (Altair ALT1250 modem) Currently T1SC is only LTE CatM

Previous versions supported the following modems:

  • Quectel module UG96 with Intel modem (3G, won’t be supported anymore in next delivery)
  • Quectel module BG96 with Qualcomm modem (LTE CatM, NBIoT and 2G Fallback)
  • Sequans module GM01Q with Sequans modem (LTE CatM)
